```mermaid
flowchart TD
    A[Walking / Running 动画播放] --> B[动画到达落脚帧]
    B --> C[Animation Event 触发]

    C --> D{左脚还是右脚?}
    D -->|左脚| E[SpawnLeftFootprint]
    D -->|右脚| F[SpawnRightFootprint]

    E --> G[CanSpawnFootprint 检查]
    F --> G

    G --> H{是否允许生成脚印?}
    H -->|否| H1[结束，不生成脚印]
    H -->|是| I[SpawnFootprint]

    I --> J[根据脚骨骼位置向下 Raycast]
    J --> K{是否命中地面?}

    K -->|否| K1[结束，不生成脚印]
    K -->|是| L[计算脚印位置和旋转]

    L --> M[计算 Walk / Run 对应的 Size 和 Offset]
    M --> N{是否使用对象池?}

    N -->|是| O[FootprintDecalPool.SpawnFootprint]
    N -->|否| P[Instantiate 新 Decal]

    O --> Q[GetAvailableInstance]
    Q --> R{available 中是否有空闲脚印?}

    R -->|有| S[从 available 取出脚印]
    R -->|没有| T{是否达到 maxFootprints?}

    T -->|没有达到| U[CreateNewInstance 创建新脚印]
    T -->|已经达到| V{是否允许复用最旧脚印?}

    V -->|是| W[从 active 中取最旧脚印复用]
    V -->|否| V1[结束，本次不生成脚印]

    S --> X[激活 Projector]
    U --> X
    W --> X
    P --> X

    X --> Y[设置 Decal 数据]
    Y --> Z[ScreenSpaceDecalProjector SetActive true]

    Z --> AA[OnEnable]
    AA --> AB[加入 ScreenSpaceDecalProjector.ActiveProjectors]

    AB --> AC[ScreenSpaceDecalFeature 执行]
    AC --> AD[遍历 ActiveProjectors]
    AD --> AE[设置 Shader 参数]
    AE --> AF[cmd.DrawMesh 绘制 Decal Volume Box]

    AF --> AG[Shader 执行]
    AG --> AH[采样 Camera Depth Texture]
    AH --> AI[重建 World Position]
    AI --> AJ[World Position 转 Decal Local Space]
    AJ --> AK{是否在 Decal Box 内?}

    AK -->|否| AK1[discard]
    AK -->|是| AL[计算 UV 并采样脚印贴图]

    AL --> AM[计算 Fade 和 Opacity]
    AM --> AN[Alpha Blend 到 Camera Color Target]
    AN --> AO[脚印显示在画面上]

    AO --> AP[生命周期开始]
    AP --> AQ[显示 visibleTime]
    AQ --> AR[淡出 fadeTime]
    AR --> AS[Release 回对象池]

    AS --> AT[从 active 移除]
    AT --> AU[SetActive false]
    AU --> AV[OnDisable]
    AV --> AW[从 ActiveProjectors 移除]
    AW --> AX[放回 available 等待复用]
```
